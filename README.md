# Meus Estudos de Programação 🚀

Este repositório serve para organizar os meus aprendizados em **Python** e o desenvolvimento do meu projeto de e-commerce.

## 🛒 Projeto Supermercado & Açougue
Um sistema para facilitar pedidos de carnes e produtos de mercado em Ibaiti, com foco em:
* Cadastro de preços por quilo.
* Cálculo automático de valores.
* Integração para pedidos via WhatsApp.
